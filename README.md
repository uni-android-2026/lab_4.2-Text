# Практическая работа №4
## Задание: Разработка приложения
Концепция приложения: экран с длинным форматированным текстом и изображением, требующий прокрутки.

## Проделанная работа:
1. Создана разметка с `ScrollView` и вертикальным `LinearLayout` для прокручиваемого контента.
2. Добавлено изображение (`bird.png`) в `ImageView` с режимом `centerCrop`.
3. Длинный текст вынесен в `strings.xml`, использованы переносы строк и базовое форматирование (`<b>`, `<i>`, `<u>`).
4. Настроены размеры текста, чтобы контент читался на экране.

## Итоги
Получился экран с прокруткой, иллюстрацией и форматированным текстом, закрепляющий работу со `ScrollView` и строковыми ресурсами.
